# server-layout
A place to brain dump all my server layouts, plans, and projects

## Some fun "retro" Windows programs
- [MonitorES](https://github.com/ukanth/monitores) - allows screensaver to activate whenever you lock your desktop
- [RetroBar](https://github.com/dremin/RetroBar) - Replaces taskbar with a classic style (lots of cool configuration options)
- [Winaero Tweaker](https://winaerotweaker.com/) - used to change system font, startup sounds, etc.
- [OpenShell](https://github.com/Open-Shell/Open-Shell-Menu) - Replaces start menu with a classic style (lots of cool configuration options)
